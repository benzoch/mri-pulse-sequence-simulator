About MRI Pulse Sequence Simulator

This MRI Pulse Sequence Simulator provides a real-time visualization of K-space reconstruction as the pulse sequence progresses, using open-source pulse sequences from the Pulseq Project. For detailed code examples and specifications, see the official documentation in the External Resources section.



The simulator offers the following features:

* Basic pulse sequence simulator
* Image reconstruction from k-space data
* Support for over 13 sequence types
* SNR (Signal-to-Noise Ratio) calculation
* Timeline visualization of sequence events
